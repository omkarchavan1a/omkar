# **App Name**: microGT Services Showcase

## Core Features:

- Service Category Browsing: Browse available services by category (Data, Video Editing, Presentations).
- Service Detail View: View detailed information and examples for each service.
- AI Project Matching: Use an AI tool to recommend services based on user project descriptions and past project data.
- Inquiry Form Submission: Submit project inquiry forms that POST to webhook https://hooks.zapier.com/hooks/catch/25226128/usoq546/.
- Contact Information Access: Access agency contact information and a contact form.

## Style Guidelines:

- Primary color: Deep blue (#293B5F) to convey professionalism and trust.
- Background color: Light gray (#E5E7EB), providing a clean and modern backdrop.
- Accent color: Subtle green (#A3B18A) for highlighting key interactive elements and CTAs.
- Body and headline font: 'Inter' (sans-serif) for a clean, modern, objective feel.
- Use clean, minimalist icons representing each service category.
- Employ a grid-based layout for a structured and professional appearance.
- Subtle transitions and animations on page load and when interacting with UI elements.
